API Guide 

API & Services Used

AQICN

Website : aqicn.org

The World Air Quality Index project is a non-profit project started in 2007. Its mission is to promote air pollution awareness for citizens and provide a unified and world-wide air quality information. The project is providing transparent air quality information for more than 130 countries, covering more than 30,000 stations in 2000 major cities, via those two websites: aqicn.org and waqi.info.

Google Maps API

website : https://cloud.google.com/maps-platform/

We have used google maps javascript api to generate the maps with the markers. The code snippets for the front end are provided on the documentation pages. The data that goes in the markers is retrieved from the database and the styling of the marker is done using css, Javascrript.

Leaflet Maps

website : https://leafletjs.com

Leaflet a js library for responsive maps is used along with the waqi.info and aciqn.org/api to present the air quality index on the maps. The main reason to use this library is its size. It weighs only 38KB of js for its impressive design.

AWS Elastic BeanStalk

website : https://aws.amazon.com

The application is hosted on AWS Elatic Beanstalk with the underlying environment containing a server run on python. Using Elastic Beanstalk is more efficient for its quick deployment, and the environment is completely provided by AWS.

AWS RDS

website : https://aws.amazon.com

The application is utilizing the backend database on AWS RDS. The python Flask server connects the database upon every request, runs the query and returns back the response to the front-end.
